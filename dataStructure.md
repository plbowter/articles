# 数据结构
1.哈希（Hash）
哈希是干嘛用的?看下面的伪代码，理解哈希的定义：

```
a <- {
  '0':0,
  '1':1,
  'key':'value',
  '键':'值'
}
```
一个 `key` 对应一个 `value`。所有满足一个键、一个值的这种结构，都叫哈希。比如 http 请求的第二部分和 http 响应的第二部分了解了哈希的概念，那哈希怎么用呢？
